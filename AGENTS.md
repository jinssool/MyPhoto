# AGENTS.md

## Project Rules

- Build a family photo memory album, not a storage product.
- Google Drive is the original photo storage.
- The web app is an image-centered browsing UI.
- MVP assumes one shared family account.
- Keep UI simple, warm, and parent-friendly.

## Do Not Implement In Task 0

- Google Drive API integration.
- Supabase integration.
- Face recognition.
- Comments, voice memos, payments, invitations, or complex permissions.
- Any Drive delete, rename, or move behavior.

## Safety

Never delete, rename, move, or otherwise mutate original Google Drive files. Cleanup should only mean app-level visibility or review states in later tasks.
